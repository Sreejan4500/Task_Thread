# Notes: Cancellation in PLINQ (Professional)

*From `5. Cancellation in PLINQ.txt`.*

---

## 1. Wiring cancellation into a PLINQ query

PLINQ supports cancellation via `WithCancellation(cts.Token)`, applied after `AsParallel()`. Cancellation may be triggered from within the query/consumer pipeline (e.g., based on observed items).

---

## 2. Non-determinism and observation

Because execution is parallel and unordered, the cancellation trigger condition can be observed on higher values early (e.g. hitting `20` before `8`), so cancellation is not tied to a predictable sequence.

---

## 3. Exception surface

The video demonstrates cancellation surfacing as `OperationCanceledException` (distinct from the aggregate exception patterns used for multiple faults).

---

## 4. Example shape

```csharp
using var cts = new CancellationTokenSource();

try
{
    Enumerable.Range(1, 200)
        .AsParallel()
        .WithCancellation(cts.Token)
        .ForAll(x =>
        {
            if (x > 8)
                cts.Cancel();
        });
}
catch (OperationCanceledException ex)
{
    Console.WriteLine(ex.Message);
}
```
