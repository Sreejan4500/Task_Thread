# Notes: Which thread is used (Professional)

*From `3. Which thread is used.txt`.*

---

## 1. Startup runs on the calling thread

Before the first **await**, the async method executes on the **calling thread** (e.g. main thread / UI thread). This avoids unnecessary thread switches for the initial synchronous portion.

---

## 2. Continuations may run on a different thread

After an **await**, the remainder of the method runs as a **continuation**. In a console app (no special context), continuations commonly run on a **thread-pool** thread. The pool may reuse the same worker thread for multiple continuations.

---

## 3. Execution order with await

The observed order is:

- Run code up to the await in `Main`
- Enter `FetchDataAsync` and run until its await
- When the awaited task completes, run the continuation in `FetchDataAsync`
- Then run the continuation in `Main`

---

## 4. Example code: printing thread IDs

```csharp
class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine($"1. Main thread id:{Thread.CurrentThread.ManagedThreadId}");

        Console.WriteLine("Starting to do work.");
        var data = await FetchDataAsync();
        Console.WriteLine($"Data is fetched: {data}");

        Console.WriteLine($"2. Thread id:{Thread.CurrentThread.ManagedThreadId}");

        Console.WriteLine("Press enter to exit.");
        Console.ReadLine();
    }

    static async Task<string> FetchDataAsync()
    {
        Console.WriteLine($"3. Thread id:{Thread.CurrentThread.ManagedThreadId}");

        await Task.Delay(2000);

        Console.WriteLine($"4. Thread id:{Thread.CurrentThread.ManagedThreadId}");

        return "Complex data";
    }
}
```

---

## Summary table

| Concept | Practical meaning |
|--------|-------------------|
| **Before first await** | Runs on the caller thread. |
| **After await** | Continuation runs on a pool thread in console apps; pool may reuse threads. |
| **Why this matters** | Better responsiveness and resource usage without forcing a dedicated thread. |
