# Notes: ParallelLoopResult (Professional)

*From `6. ParallelLoopResult.txt`.*

---

## 1. Result object returned by Parallel loops

`Parallel.For`/`ForEach` return a `ParallelLoopResult`, available after the loop completes (parallel loops are blocking calls).

---

## 2. Interpreting IsCompleted and LowestBreakIteration

- `IsCompleted == true` indicates all iterations completed.
- `IsCompleted == false` indicates early termination (break/stop/exception).
- `LowestBreakIteration`:
  - set when `Break()` occurs (lowest requested break index)
  - `null` when `Stop()` occurs or when termination is due to exceptions (so you distinguish exception vs stop using the surrounding `try/catch`).

---

## Summary table

| Field | Meaning |
|------|---------|
| `IsCompleted` | Completed all iterations or terminated early. |
| `LowestBreakIteration` | Break point (if `Break()` was used). |
